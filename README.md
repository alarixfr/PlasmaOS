# PlasmaOS
PlasmaOS is a concept os made in html, css and js
This project is for made for fun.

![PlasmaOS Logo](./assets/img/lockscreen.png)

# Installation
0. Install VS Code or any other code editor and install git.
1. Clone:
```
git clone https://github.com/alarixfr/plasmaos
```

2. Run using live server or manually by opening the index.html file path in your browser.
